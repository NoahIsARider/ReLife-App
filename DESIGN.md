# ReLife - 设计语言

## 产品灵魂

**意象锚点**：清晨六点的白色花园，玫瑰灰的晨光透过薄雾洒在石径上，露珠悬在花瓣边缘，一切都安静、温柔、充满尊严。这不是告别的地方，而是"把重要的事好好交代"的地方。

**核心情绪**：不是悲伤，是"认真地爱过，认真地安排"。像一个温暖的拥抱，告诉你"一切都安排好了，你放心"。

## 气质与意象

- **关键词**：温柔庄严 / 莫兰迪诗意 / 生命花园 / 高级克制
- **具象场景**：一位优雅的法国女性在晨光中的书房写信，桌上放着干花和一杯伯爵茶。她用钢笔在质感厚重的纸上写下给家人的信。窗外是玫瑰花园。一切从容、有序、充满爱。
- **反面意象（禁止）**：墓碑、黑色、十字架、蜡烛、枯萎的花、医院、悲伤的脸

## 视觉策略

- **摄影/图像方向**：柔和自然光、浅景深、暖灰调滤镜。植物元素以"生长中的花"而非"枯萎的花"出现——象征生命的延续
- **图形语言**：圆润的有机形态、极细的线条、留白呼吸感。图标使用细线条风格，不厚重
- **装饰元素**：花瓣飘落的抽象线条、柔和的渐变光晕、极细的金色分隔线

## 配色方案

### 主色系统（莫兰迪玫瑰）

| 名称 | 色值 | 意象来源 |
|------|------|----------|
| **Rose Dust** 主色 | `#C4A0A0` | 干燥玫瑰花瓣的灰粉色，不鲜艳但持久 |
| **Rose Mist** 主色浅 | `#D4B5B5` | 晨雾中玫瑰的轮廓 |
| **Rose Shadow** 主色深 | `#9E7B7B` | 傍晚玫瑰投下的影子 |
| **Warm Ivory** 根背景 | `#F5F0EB` | 温暖象牙白，像旧信纸 |
| **Soft Linen** 卡片背景 | `#FAF7F4` | 亚麻白，微微透出暖意 |
| **Cloud Gray** 次级背景 | `#EDE8E3` | 云朵灰，用于凹陷区域 |

### 辅助色

| 名称 | 色值 | 用途 |
|------|------|------|
| **Sage Green** | `#A8B5A0` | 安全/已完成状态 |
| **Dusty Gold** | `#C4B08B` | 强调/重要标记 |
| **Slate Blue** | `#8E9AAB` | 信息/中性提示 |
| **Warm Brown** 文字主 | `#4A3F3A` | 正文文字，暖棕黑 |
| **Dusty Text** 文字次 | `#8B7D75` | 辅助文字 |
| **Muted Placeholder** | `#BEB5AD` | 占位符 |

### 色彩禁忌
- 禁止使用纯黑 `#000000`，最深色用 `#4A3F3A`
- 禁止使用纯白 `#FFFFFF` 做卡片，用 `#FAF7F4`
- 禁止使用高饱和度色彩，所有颜色必须带灰调
- 禁止红色用于"错误/危险"场景，改用柔和的 `#C48A8A`

## 字体排版

- **中文标题**：系统默认（fontWeight: 700/800），追求克制优雅
- **英文/数字**：系统默认，数字使用等宽感
- **排版节奏**：
  - 页面大标题：28px, fontWeight 800, 字间距 +1
  - 区块标题：20px, fontWeight 700
  - 卡片标题：16px, fontWeight 600
  - 正文：14px, lineHeight 22
  - 辅助文字：12px, color: Dusty Text
  - 数据大字：32px, fontWeight 800, color: Rose Dust

## 动效与交互

- **运动性格**：缓慢、柔和、有呼吸感。像花瓣飘落，不是弹跳
- **缓动曲线**：`Easing.bezier(0.25, 0.1, 0.25, 1.0)` —— 柔和的 ease-in-out
- **卡片入场**：从下方 20px 处淡入，duration 400ms
- **按压反馈**：scale 1 → 0.98, duration 150ms，轻柔不突兀
- **状态切换**：颜色渐变 300ms，无突变

## 核心样式手法

### 卡片：柔和阴影 + 大圆角
```
borderRadius: 20
backgroundColor: '#FAF7F4'
shadowColor: '#4A3F3A'
shadowOffset: { width: 0, height: 2 }
shadowOpacity: 0.06
shadowRadius: 12
```

### 图标容器：圆形 + 主色淡底
```
width: 48, height: 48, borderRadius: 24
backgroundColor: 'rgba(196, 160, 160, 0.12)'
```

### 主按钮：柔和渐变
```
colors: ['#C4A0A0', '#D4B5B5']
borderRadius: 16
paddingVertical: 16
```

### 标签胶囊
```
backgroundColor: 'rgba(196, 160, 160, 0.10)'
borderRadius: 9999
paddingHorizontal: 14, paddingVertical: 6
```

## 设计禁忌（绝对不做）

- 不要墓碑、十字架、蜡烛、黑色花圈等死亡符号
- 不要用纯黑/纯白的高对比度
- 不要用红色做警告（用柔和的玫瑰红）
- 不要用"死亡""去世"等刺眼词汇做UI文案，用"离开""远行""交接"
- 不要让整体氛围压抑，要保持"阳光透过窗帘"的通透感
- 不要用尖锐的直角，所有容器至少 12px 圆角
- 不要用高饱和度色彩打破宁静感
